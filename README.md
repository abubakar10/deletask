# Deletask Frontend Redesign

Modern, conversion-focused React landing page for Deletask built with Vite + Tailwind CSS.

## Tech stack

- React
- Vite
- Tailwind CSS (via `@tailwindcss/vite`)

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run preview` - preview the production build locally
- `npm run lint` - run ESLint checks

## Netlify deployment

This project is ready for Netlify:

- `netlify.toml` sets build command and publish directory
- `public/_redirects` enables SPA routing fallback
